GRUNDLÆGGENDE WEB

Temabeskrivelse

I temaet Grundlæggende web får vi stiftet bekendtskab med HTML og CSS, som vi skulle bruge til at lave vores første responsive website. Vi fik tildelt et layout diagram som vi skulle følge og kode efter, således vores grids og kolonner blev delt op som de skulle. Sitet skulle følge følge en designstil som vi fik tildelt, det var i mit tilfælde flat design. Flat design har referencer til “Swiss Design” og “Bauhaus”, og kendetegnes ved det flade 2D design og lette farver. Til sidste blev vi introduceret til photoshop, som vi brugte til at kreere til splash billede til vores website. Da mit website omhandler stueplanter lavede jeg et splash billeder med planter, jeg søgte inspiration på google og plante hjemmesider.
